# ProxBot v0.3.0

## Espanol

### Resumen

ProxBot v0.3.0 se centra en monitorizacion automatica y alertas controladas para
homelabs.

### Objetivo

Pasar de diagnostico manual a comprobaciones programadas sin spam.

### Cambios planeados

- Configuracion `monitoring` en `config.json`.
- Motor de monitorizacion.
- Alertas por cambios de estado.
- Ultimo diagnostico guardado.
- Comando `/monitor`.
- Comando `/ultimodiagnostico`.

### Limitaciones

- No sustituye a Uptime Kuma.
- No incluye GUI.
- No incluye base de datos.
- No ejecuta acciones destructivas.
- No controla Proxmox ni otros servicios externos.

## English

### Summary

ProxBot v0.3.0 focuses on automatic monitoring and controlled alerts for
homelabs.

### Goal

Move from manual diagnostics to scheduled checks without alert spam.

### Planned changes

- `monitoring` configuration in `config.json`.
- Monitoring engine.
- State-change alerts.
- Saved last diagnostics.
- `/monitor` command.
- `/ultimodiagnostico` command.

### Limitations

- Does not replace Uptime Kuma.
- No GUI.
- No database.
- No destructive actions.
- No Proxmox or external service control.
