# Release v0.9.0 — Inventario detectado desde Proxmox VE

**Fecha:** 2026-05-05
**Tag:** [v0.9.0](https://github.com/Caarrasco22/proxbot/releases/tag/v0.9.0)

## Resumen

Esta version añade la capacidad de consultar el inventario real de VMs y contenedores desde Proxmox VE, con cache local opcional y sin modificar el inventario manual del homelab.

## Novedades

- **Nuevo comando `/proxmox-inventario`**: consulta el inventario real de tu Proxmox VE con las opciones `ver` (en tiempo real) y `cache` (desde cache local).
- **Cache local opcional**: guarda el ultimo inventario consultado en `data/proxmox-inventory-cache.json` para consultas rapidas sin depender de la API de Proxmox.
- **Boton Proxmox en `/panel`**: acceso directo al inventario desde el panel principal, con fallback automatico a cache si la API no responde.
- **Seguridad**: proteccion contra path traversal en rutas de cache y validacion de `inventoryCachePath` en `utils/validateConfig.js`.
- **Solo lectura**: no modifica `config.json` ni el inventario manual del homelab.
- **Documentacion**: guias nuevas en `docs/PROXMOX-INVENTORY.md` y `docs/PROXMOX-INVENTORY.en.md`.

## Cambios tecnicos

- Añadidas utilidades `getProxmoxInventoryResources`, `readInventoryCache`, `writeInventoryCache` y `resolveInventoryCachePath` en `utils/proxmox.js`.
- Actualizados `README.md` y `README.en.md` con badges de shields.
- Version bump a `0.9.0` en `package.json`.
